# Wms

A core library for [WMS](https://github.com/phoorichet/wms).

## Installation

Add this line to your application's Gemfile:

    gem 'wms'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wms

## Usage

This gem is included in WMS.
